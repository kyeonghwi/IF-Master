# IF-Master 프로젝트 컨텍스트

## 지원 회사
노아에이티에스 2026년 공채 15기 2차 신입사원 채용 (IT개발 / 금융시스템 개발)
- AI(LLM) 기반 바이브코딩 방법론 도입 중
- 개발 포트폴리오 필수 제출 (기획서 + 개발문서 + 동작하는 웹앱)
- 2개 이상 제출 시 가산점

## 프로젝트 구성
1. **보험사 금융 IT 인터페이스 통합관리시스템** (1번 - 완료)
   - 기획서: `01_기획서.md`
   - 개발문서: `02_개발문서.md`
   - 핵심 기능: 통합 모니터링 대시보드, 로그 조회, 원클릭 재처리(멱등성 보장)
   - 스택: FastAPI(Python) + Next.js + PostgreSQL + Render/Vercel
   - AI 도구: Antigravity, Claude Code, Gemini 3.1 Pro

2. **원가/관리회계** (2번 - 진행 예정)
   - 가산점용 두 번째 프로젝트
   - 금융 수학 배경 없음 → 백엔드 로직(집계, 배분, 원가 계산) 중심으로 구현

## 기획서 작성 규칙
- 기대 효과는 "정성적"으로만 (수치 만들어내지 않기)
- 바이브코딩 AI 도구: Antigravity, Claude Code, Gemini 3.1 Pro
- 배포: Vercel(프론트) + Render(백엔드 + DB)
- 스택: FastAPI(Python) + Next.js + PostgreSQL

## 사용자 배경
- SW Campus 프로젝트 경험 (Spring Boot, Next.js, PostgreSQL, DDD, 멀티모듈)
- 금융 수학 배경 없음
- 빠른 프로토타이핑 선호 → Python FastAPI
